# [FAQ]
Q: 65세 이상 환자 명단
A: SELECT subject_id, gender, anchor_age, anchor_year, dod FROM patients WHERE anchor_age >= 65

Q: 최근 1년간 사망한 환자 명단 (환자ID, 사망일자)
A: SELECT subject_id, dod FROM patients WHERE dod IS NOT NULL AND dod >= ADD_MONTHS(SYSDATE, -12)

Q: 80세 이상 환자 명단 (환자ID, 성별, 나이)
A: SELECT subject_id, gender, anchor_age FROM patients WHERE anchor_age >= 80

Q: 최근 1년간 사망한 환자 명단(환자ID, 사망일자, 성별)
A: SELECT subject_id, dod, gender FROM patients WHERE dod IS NOT NULL AND dod >= ADD_MONTHS(SYSDATE, -12)

# [유의어]
"여자", "여성" → gender='F'
"남자", "남성" → gender='M'
"나이", "연령", "만" → anchor_age
"사망", "사망일" → dod
"입원", "입원ID" → hadm_id
"퇴원", "퇴원ID" → dischtime
